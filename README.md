# Sistema de Controle de Biblioteca

<p>
O sistema permite o cadastro e gerenciamento de livros. Com as seguintes funcionalidades: 

</p>
<p>
<li>Cadastrar livro</li>
<li>Consultar livro</li>
<li>Alterar dados</li>
<li>Remover livro</li>
<li>Listar todos</li>
<li>Realizar empréstimo</li>
<li>Realizar devolução</li>
<li>Sair</li>
</p>
